# Telegram Moderation Bot for Vercel

Этот проект — готовый к деплое Telegram-бот на Vercel, который:
- Удаляет сообщения с признаками "18+".
- Удаляет сообщения про гадания / предсказания (таро, гороскоп и т.п.).
- Отправляет предупреждение упоминающему пользователя (например, `@username`) после удаления.

> ВАЖНО: **Никогда не вставляй токен в код**. Токен должен храниться в переменных окружения Vercel.

## Файлы в проекте
- `api/index.py` — основной код бота (Webhook handler для Vercel).
- `requirements.txt` — зависимости.
- `vercel.json` — конфигурация функций.
- `.gitignore` — стандартные исключения.
- `README.md` — это руководство.

## Деплой на Vercel
1. Создай новый репозиторий на GitHub и закоммить файлы из этого архива.
2. На Vercel выбери **Import Git Repository** и подключи репозиторий.
3. В настройках проекта Vercel → **Environment Variables** добавь:
   - `BOT_TOKEN` = `<твой_новый_токен>` (токен, который ты получил у BotFather).
4. Деплой проекта.
5. После успешного деплоя установи webhook:
   ```
   https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/setWebhook?url=https://<твой_домен>.vercel.app/api
   ```
   Пример:
   ```
   https://api.telegram.org/bot123456:ABCDEF/setWebhook?url=https://mybot.vercel.app/api
   ```
6. Добавь бота в группу и выдай ему права администратора с разрешением *Delete messages*.

## Примечания и доработки
- В коде есть список регулярных выражений. Ты можешь расширить их, если нужно блокировать другие фразы.
- По умолчанию сообщения администраторов группы не удаляются.
- Если хочешь — могу добавить:
  - логирование удалённых сообщений в файл или Google Sheets,
  - команду `/set_warn on|off`,
  - белый список пользователей.

Если хочешь, я могу сразу помочь с созданием нового токена и пошаговой настройкой вебхука.
